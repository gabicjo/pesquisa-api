# O que é isso?
Essa é uma api que tem apenas um metodo POST, que recebe um json com 19 respostas de um (Formulario de disgnostico de problemas auditivos)[https://github.com/gabicjo/formulario], que envia os dados que o usuario marcou para serem armazenadas em um banco de dados simples usando sqlite.
O objetivo dessa api é justamente guardar esses dados para posteriormente serem analizados.
# O que eu aprendi?
- Criar APIs usando o framework Flask
- Criar rotas com diferentes metodos
- Organização de pastas de uma api
- Adicionar dados de um json para um banco de dados local
- Utilizar SQL + sqlite3 para armazenar dados localmente
# Como usar?
1. Instale o python no seu computador (se ainda não tiver)
2. Abra um terminal na pasta do projeto
3. Digite no terminal `python3 server.py` (se estiver no linux, pode ser `bg python3 server.py` para rodar em segundo plano.
